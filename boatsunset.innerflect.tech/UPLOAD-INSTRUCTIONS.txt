OPTION 1: Upload via ZIP (RECOMMENDED)
--------------------------------------
1. Compress this "upload" folder to a ZIP file on your computer
2. Log in to cPanel → File Manager
3. Go to public_html (or your domain root folder)
4. Upload the ZIP file
5. Right-click the ZIP → Extract
6. Delete the ZIP after extraction
7. Done. Site is live.

OPTION 2: Direct upload
--------------------------------------
1. Log in to cPanel → File Manager
2. Go to public_html (or your domain root folder)
3. Select ALL files and folders inside this "upload" folder
4. Drag and drop them into public_html
5. Wait for upload to finish
6. Done. Site is live.

⚠️  Upload the CONTENTS of this folder into public_html,
    NOT the "upload" folder itself (so index.html is at
    yourdomain.com/index.html, not yourdomain.com/upload/index.html).
